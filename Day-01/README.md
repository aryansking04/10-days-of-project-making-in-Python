# 🧮 Day 1: SmallCalci (Python Calculator)

Welcome to **Day 1** of my **#10DaysOfProjectMaking** challenge 🚀
Today’s project is a simple command-line calculator built using Python.

---

## 📌 Project Overview

**SmallCalci** is a basic calculator that performs fundamental arithmetic operations:

* Addition ➕
* Subtraction ➖
* Multiplication ✖️
* Division ➗

This project is intentionally kept simple to strengthen core Python concepts like:

* User input handling
* Conditional statements
* Functions
* Error handling

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Functions
  * Conditional Logic (`if-else`)
  * Exception Handling

---

## ⚙️ Features

* Takes user input from the terminal
* Supports basic arithmetic operations
* Handles invalid inputs (like division by zero)
* Simple and beginner-friendly interface
